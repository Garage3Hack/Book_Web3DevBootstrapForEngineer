# ブロックチェーンサンプル（5章）　 NFTコントラクト（ERC-721）の開発

## リスト2. Hardhatによるスマートコントラクトのコンパイル
```
npx hardhat compile
```

## リスト4. Hardhatによるテスト実行
```
npx hardhat test
```

```
npx hardhat node
```

## リスト7. スマートコントラクトのデプロイ
```
npx hardhat run --network localhost scripts/deploy-local.ts
```
